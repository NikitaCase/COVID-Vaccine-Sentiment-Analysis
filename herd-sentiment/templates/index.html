Introduction

This project analyzes tweets about Covid 19 Vaccines.
Tweets were gathered from the Twitter API which mentioned: Covid Vaccine, Covid-19 Vaccine, Moderna, Pfizer, Pfizer-BioNTech, AstraZeneca, Astra Zeneca
Tweets were cleaned and analyzed for polarity and subjectivity

A more thorough methodology can be found in the readme here.
