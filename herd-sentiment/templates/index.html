    • Introduction

    This project analyzes tweets about Covid Vaccine

    Methods

    1. API keys were obtained from Twitter
    2. API was accessed using Python and Tweepy to collect all tweets which mentioned the following:
      Covid Vaccine, Covid-19 Vaccine, Moderna, Pfizer, Pfizer-BioNTech, AstraZeneca, Astra Zeneca
      This data excluded retweets.
    3. Tweets were cleaned and sorted using Pandas and exported to csvs and an SQL database
    4. Tweets were analysed for sentiment using someting or the other
    5. Aggregation was performed on the analyzed tweets to create visualizations

    Jansen Vaccine not Included in final list because it was approved on March 5th and data collection started on March 7th.
    More thorough methodology can be found in the readme here.
